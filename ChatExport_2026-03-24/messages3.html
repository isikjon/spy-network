Для реализации автоплатежей с карты или оплаты в "один клик" (когда пользователь вводит данные карты только один раз), потребуется выполнить следующие шаги:

1. Проверьте поддержку автоплатежей в вашей CMS.
Убедитесь, что система управления сайтом (CMS), на которой построен ваш сайт, поддерживает автоплатежи. При необходимости уточните этот вопрос в службе поддержки вашей CMS.

2. Предоставьте скриншоты с описанием отвязки карты.
Необходимо сделать один или несколько скриншотов вашего сайта, чтобы было понятно, как пользователь может самостоятельно отвязать карту. На скриншотах должен быть виден URL вашего сайта.

Сценарий отвязки карты должен позволять пользователю удалить привязку в любой момент без необходимости отправлять запросы.
Функционал отвязки должен быть доступен в личном кабинете вашего сайта, а при удалении карта должна исчезнуть из системы.
3. Укажите планируемые обороты.
Сообщите нам предполагаемые месячные обороты по онлайн-платежам для согласования.

4. Укажите короткое имя магазина для банковской выписки.

Формат: 14 символов («YM*» + ещё 11). Допустимы латиница, цифры и знаки «.», «-», «_», « »;
Называние должно быть информативным и связано с наименованием вашего проекта (приложения/бота) — отражать его суть, быть созвучным или содержать ключевые элементы названия;
5. Реализуйте техническую часть подключения.
Подробная информация о методах реализации доступна по ссылке.

Вопросы по интеграции можно отправить на электронную почту b2b_support@yoomoney.ru.

Обратите внимание:
После подключения автоплатежей на форме оплаты появится чекбокс «Запомнить данные карты».
Реализация процедуры отвязки карты обязательна: при удалении карты вы должны удалить токен из своей системы. Сообщать нам об отвязке не требуется.
